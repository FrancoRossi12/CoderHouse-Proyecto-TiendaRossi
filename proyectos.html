Proyecto 1 - Juego en Python

Este proyecto consistió en la creación de un juego sencillo en Python utilizando la librería Pygame. El objetivo del juego es que el jugador controle un personaje que debe esquivar obstáculos y recolectar objetos para ganar puntos. Se implementó una mecánica de niveles y un sistema de puntuación. Además, se utilizó un archivo de configuración externo para ajustar la dificultad y la apariencia del juego.

Proyecto 2 - Sitio web en HTML y CSS

Este proyecto consistió en la creación de un sitio web estático utilizando HTML y CSS. El sitio es una página de presentación de una empresa ficticia que ofrece servicios de diseño y desarrollo web. Se utilizó un diseño responsive para que el sitio se adapte a distintos tamaños de pantalla. También se incluyeron animaciones sencillas utilizando CSS.

Proyecto 3 - Programa en C++

Este proyecto consistió en la creación de un programa en C++ que realiza operaciones matemáticas básicas con números complejos. Se creó una clase para representar números complejos y se sobrecargaron los operadores de suma, resta, multiplicación y división para que puedan operar con números complejos. Se implementó una función para imprimir los números complejos de manera ordenada. El programa también permite al usuario ingresar los números complejos y las operaciones a realizar.
